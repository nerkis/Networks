WebServer

Last updated: 2/16/12

A multithreaded web server that displays the browser's HTTP request message. To run the server and view HTTP request messages, use terminal to navigate to the erkisPA2 folder, compile WebServer.java (javac Webserver.java) and then run WebServer (java Webserver). Open a browser and type in: <http://searles-224-06.local:5020/> (replace hostname and port number as necessary).
